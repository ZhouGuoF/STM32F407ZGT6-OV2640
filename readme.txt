本项目是大三的物联网课设期间做的，由于时间，硬件等条件，不足的地方不少，勿怪。



实验器材:
	STM32F407ZGT6+ov2640
	

	
硬件资源:

		OV2640模块 ------------ STM32开发板
		 OV_D0~D7  ------------  PE6/PE5/PB6/PC11/PC9/PC8/PC7/PC6
		 OV_SCL    ------------  PD6
		 OV_SDA    ------------  PD7
		 OV_VSYNC  ------------  PB7
		 OV_HREF   ------------  PA4
		 OV_RESET  ------------  PG15
		 OV_PCLK   ------------  PA6
		 OV_PWDN   ------------  PG9 
		 
上位机是用Qt写的，文件名为Car_Server ，使用了MQTT协议连接阿里云物联网平台。

程序流程：
   stm32f407zgt6驱动ov2640采集图像数据，使用esp8266发送数据给上位机，Qt上位机接收数据，并将每帧图像数据转换
为opencv的Mat类型，进行图像分析，获得路线的位置，从而控制车的方向，并上传行驶方向数据到阿里云物联网平台实时显示。
